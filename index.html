Introduction

Java Cart is a Java based system created to give the user an inteface where the user can select orders from a menu to be added or deleted from their cart. The system will also display final cost including taxes.

Appetizers

The System

The System is our abstract class. An abstract class is used to provide a base for subclasses to extend and implement the abstract methods and override or use the implemented methods in abstract class. This is where our inventory can be created and updated.

Main Courses

Main Entry Point

The Main Entry Point is the main method and the entry point of our system. This is where the Menu is to add, display, remove or quit.
